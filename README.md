# Fraud Detection App

A Flask web app for credit card fraud detection using the `creditcard.csv` dataset.

## Run locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the app:

```bash
python3 app.py
```

3. Open:

```text
http://127.0.0.1:5000
```

## Create a downloadable ZIP

Run:

```bash
./scripts/create_zip.sh
```

This creates:

```text
dist/fraud_detection_app.zip
```

The ZIP includes the app code, templates, static files, dataset, and setup instructions.
